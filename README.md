# garmadoro

a pomodoro timer for my garmin forerunner 165

## Compatibility

- garmin forerunner 165

can be expanded to support other garmin watches easily just needs product id change w/ minor tweaks

## Features

- 25 min work sessions, 5 min short breaks, 30 min long break after 4 pomodoros
- vibration alerts at the end of each session
- optional tick vibration during work sessions
- configurable via garmin connect iq settings

## Usage

Start/Stop starts a session. pressing it again will reset/exit
