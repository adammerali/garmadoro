# garmadoro

a pomodoro timer for the garmin forerunner 165. built to have a quick study/focus session right from the watch.

## compatibility

- garmin forerunner 165

can be expanded to support other garmin devices in the future.

## features

- 25 min work sessions, 5 min short breaks, 30 min long break after 4 pomodoros
- vibration alerts at the end of each session
- optional tick vibration during work sessions
- configurable via garmin connect iq settings

## usage

press the action button to start a session. press again to restart or exit.
